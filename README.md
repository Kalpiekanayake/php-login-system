# PHP Login System

This is a simple PHP login system with MySQL database, user registration, login, session management, and logout functionality.  

## Features
- User registration with password hashing
- User login with session management
- Dashboard after login
- Logout functionality
- Simple CSS styling

## Requirements
- XAMPP / WAMP / MAMP
- PHP 8+
- MySQL

## Usage
1. Clone the repository
2. Import `php_login_system.sql` in phpMyAdmin (database + users table)
3. Open `register.php` in browser to create a user
4. Login using `login.php`
